Formatting your file to upload for visualization

    First, the file needs to contain a header containing the GSMs in order

  • This takes the form of a # followed by a tab then the GSMs separated by tabs
  • i.e. "#      GSM1      GSM3      GSM45      GSM2      GSM90"

    The file will also need to contain the actual data you wish to visualize

  • This takes the form of a gene_id followed by a tab then the values measured separated by tabs
  • i.e. "vash1      0.987      0.123      0.3413      0.321      0.9832"

    Finally, including attribute data allows a more detailed view at samples

  • This takes the form of attribute_ followed by the attribute name, a tab, then the values separated by tabs
  • i.e. "attribute_gender      male      male      male      female      male"
